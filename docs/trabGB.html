1. Contextualização do Projeto

Projeto: Sistema de Monitoramento de Saúde Portátil

A UniEmbedded Solutions é especializada no desenvolvimento de sistemas embarcados de alto desempenho. Nossos produtos são usados em diversos setores, incluindo automação industrial, eletrônicos de consumo, automotivo, saúde e muito mais.

Objetivo do Projeto

Objetivo Geral: Desenvolver um sistema embarcado portátil que possa monitorar diferentes parâmetros de saúde, como frequência cardíaca, temperatura corporal e atividade física. Essas informações devem ser facilmente acessíveis ao usuário e, opcionalmente, compartilháveis com profissionais de saúde.

2. Requisitos

Requisitos

Desenvolver um sistema embarcado portátil que possa monitorar diferentes parâmetros de saúde, como frequência cardíaca, temperatura corporal e atividade física. Essas informações devem ser facilmente acessíveis ao usuário e, opcionalmente, compartilháveis com profissionais de saúde.

3. Premissas e Restrições

3.1. Premissas

Premissas feitas durante o desenvolvimento incluem:

  • usuários terão acesso a uma conectividade à internet confiável para permitir o monitoramento e controle em tempo real do sistema. Isso é crucial para garantir a transmissão eficiente de dados entre os dispositivos vestíveis e a plataforma em nuvem.
  • Os dispositivos IoT utilizados seguirão e serão compatíveis com os padrões da indústria. Isso é essencial para garantir a interoperabilidade dos dispositivos e a integração bem-sucedida com o sistema, evitando problemas de compatibilidade.
  • Os dados de consumo de energia fornecidos pelos usuários serão precisos e atualizados.

3.2. Restrições

Restrições para o desenvolvimento e implementação do projeto incluem:

  • restrições financeiras para aquisição de componentes, tecnologias e desenvolvimento de software. Essas restrições podem impactar diretamente nas escolhas de hardware e software durante o desenvolvimento.
  • O projeto deve aderir estritamente às regulamentações de saúde e privacidade de dados. Restrições relacionadas à coleta, armazenamento e transmissão de dados de saúde dos usuários devem ser cuidadosamente consideradas para garantir conformidade com normas e leis aplicáveis.
  • O sistema está sujeito a restrições de compatibilidade com tecnologias existentes. Isso inclui a necessidade de integração com plataformas de saúde digitais, dispositivos móveis e outros sistemas de monitoramento. Garantir a interoperabilidade é vital para o sucesso do projeto.

6. Desenvolvimento

6.1. Projeto de Hardware

Engenheiros de hardware devem projetar componentes físicos que sejam compactos, eficientes e seguros para o usuário. Isso inclui a seleção de sensores precisos para monitoramento de saúde, o design de uma placa de circuito eficiente e a escolha de materiais ergonomicamente amigáveis.

6.2. Projeto de Software

Desenvolver um software intuitivo que colete, processe e exiba os dados de saúde de maneira compreensível para o usuário. Isso envolve a programação de microcontroladores para a coleta de dados, a criação de interfaces de usuário amigáveis e a integração de protocolos de comunicação seguros para permitir o compartilhamento de dados com dispositivos médicos ou aplicativos de saúde.

7. Testes

7.1. Testes de Acuracidade Biométrica

Avaliação da precisão dos sensores biométricos em medir sinais vitais, como frequência cardíaca, pressão arterial e saturação de oxigênio.

7.2. Testes de Conectividade

Garantia de conectividade estável entre os dispositivos vestíveis e o aplicativo móvel associado.

7.3. Testes de Integridade de Dados

Verificação da precisão na coleta, transmissão e armazenamento dos dados de saúde.

9. Conclusão

9.1 Conclusão do Projeto

O Sistema de Monitoramento de Saúde Portátil demonstra eficácia na coleta e transmissão de dados biométricos, proporcionando aos usuários uma visão abrangente de sua saúde em tempo real. A realização dos testes evidenciou a confiabilidade das leituras e a integridade dos dados, enquanto os diagramas oferecem uma compreensão clara da arquitetura e do fluxo de informações.
